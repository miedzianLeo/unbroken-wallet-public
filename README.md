Unbroken-wallet - is a simple service which allows to track Player's money account and connected transactions. 

Assumptions:
- Player can have single account
- Player can have multiple transactions(payments) on his account
- There are two types of transactions: deposit and withdrawal - Player can deposit money to his account and 
withdraw money from his account
- Player must be verified(KYC verification) before he can make any transactions
- Only a part of the API is implemented so far - Players and accounts are created by database script. There is only
an implementation of some operations related to the transactions(payments)
